JS এর ডাটা টাইপ

Data কি?

ডাটা হলো যে কোনো ধরনের তথ্য যা একটি প্রোগ্রামে সংরক্ষণ, প্রক্রিয়াকরণ বা পরিচালনা করা যায়। যেমনঃ মানুষের নাম, জন্ম তারিখ, বয়স, মোবাইল নাম্বার এগুলো প্রতিটি একেকটি ডাটা।

Data Types বলতে কি বুঝি?

মানুষের নাম, জন্ম তারিখ, বয়স এগুলো ভিন্ন ভিন্ন ধরনের ডাটা। এই ভিন্ন ধরনের ডাটাগুলোকে বোঝার সুবিধার্থে বিভিন্ন ধাপে বিভক্ত করা হয়ে থাকে।

মূলত জাভাস্ক্রিপ্টে ২ ধরনের ডাটা টাইপ রয়েছে।

  1. প্রিমিটিভ ডাটা টাইপ (Primitive Data Type)
  2. নন-প্রিমিটিভ ডাটা টাইপ (Non-Primitive Data Type)

Primitive Data Type

প্রিমিটিভ ডাটা টাইপ হলো জাভাস্ক্রিপ্টের এমন ডাটা টাইপ, যা অবিভাজ্য এবং সরল। এ ধরনের ডাটা টাইপের মান অপরিবর্তনীয় (immutable), অর্থাৎ মান পরিবর্তন করা সম্ভব নয়। প্রিমিটিভ ডাটা টাইপ সরাসরি ভ্যালু ধারণ করে।

এগুলো বেশ কয়েক প্রকারের হয়ে থাকে। যেমনঃ

  1. String
  2. Number
  3. Boolean
  4. Undefined
  5. Null
  6. Symbol
  7. BigInt





Non-Primitive Data Type

নন-প্রিমিটিভ ডাটা টাইপ হলো জাভাস্ক্রিপ্টের এমন ডাটা টাইপ, যা অবজেক্ট-ভিত্তিক এবং জটিল ডাটা ধারণ করে। এ ধরনের ডাটা টাইপে ভ্যালু সরাসরি সংরক্ষণ করা হয় না; বরং একটি রেফারেন্স সংরক্ষণ করা হয়। এটি মিউটেবল (mutable), অর্থাৎ এর ডাটা পরিবর্তন করা যায়।

এগুলো বেশ কয়েক প্রকারের হয়ে থাকে। যেমনঃ

  1. Object
  2. Array
  3. Function
  4. Map
  5. Set

Primitive Data Type
Data Type Description Example
String টেক্সট বা বর্ণমালার ডাটা উপস্থাপন করে। এটি পরিবর্তন করা যায় না, কিন্তু নতুন string তৈরি করা যায় "Hello RRF January 2025 Batch"
Number Number বলতে সংখ্যাসূচক ডাটা বুঝায়। এটি এক ধরণের পূর্ণসংখ্যা (integer) অথবা দশমিক (float) সংখ্যা । ৪২, ৩.১৪
Boolean কেবল মাত্র দুইটি মান বা value ব্যবহার করে। এগুলোকে "true" এবং "false" দ্বারা প্রকাশ করা হয়। true, false
Undefined কোনো ভেরিয়েবল নির্ধারণ করা হয়েছে কিন্তু কোনো মান দেওয়া হয়নি, এরকম থাকলে তাকে আনডিফাইন্ড বলে। let x; // x এর মান undefined
Null ইচ্ছাকৃতভাবে কোনো মান অনুপস্থিত দেখায় বা শূন্য মান নির্দেশ করে। let y= null;
Symbol একটি একক মান (unque) তৈরি করতে ব্যবহৃত হয়। Symbol("id")
BigInt BigInt হলো জাভাস্ক্রিপ্টে একটি ডাটা টাইপ, যা খুব বড় সংখ্যাকে সঠিকভাবে উপস্থাপন করতে এবং পরিচালনা করতে ব্যবহৃত হয়। জাভাস্ক্রিপ্টের Number টাইপ সাধারণত 2^53-1 (যা প্রায় 9000 ট্রিলিয়ন) পর্যন্ত সঠিকভাবে কাজ করতে পারে। এর বেশি সংখ্যার জন্য BigInt ব্যবহার করা হয়। let bigNumber = 9007199254740991n;
Non-Primitive Data Type
Data Type Description Example
Object Object হলো একটি ডাটা টাইপ যা একাধিক মান ধারণ করতে পারে। এখানে প্রত্যেকটি মান একটি key বা property দিয়ে চিহ্নিত করা হয়। let person = { name: "Rahim", age: 40 };

এখানে naem ও age হলো key.
Array একটি লিস্ট বা ডাটাসেট, যেখানে একই জিনিসের একাধিক একাধিক ভ্যালু রাখা যায়। let fruits = ["apple", "banana", "cherry"];
Function একটি কোড ব্লক, যা নির্দিষ্ট কাজ সম্পাদন করে। এটি নিজেও একটি অবজেক্ট। প্রয়োজনে এটিকে call করে ব্যবহার করা যায়। function greet() { console.log("Hello!"); }
Map Map হলো জাভাস্ক্রিপ্টের একটি ডাটা স্ট্রাকচার, যা key-value pair আকারে ডাটা সংরক্ষণ করতে ব্যবহৃত হয়। এটি যেকোনো ধরনের কী (string, object, function, ইত্যাদি) ব্যবহার করতে পারে এবং কী-ভ্যালু সংরক্ষণের ক্রম ঠিক রাখে। এখানে বেশ কয়েক ধরনের কাজ করা যায়। let map = new Map(); মান সেট করা map.set("name", "Tanvir");
মান পড়া console.log(map.get("name"));
আউটপুট: Tanvir
Set Set হলো জাভাস্ক্রিপ্টের একটি ডাটা স্ট্রাকচার, যা একাধিক unique values সংরক্ষণ করে। এতে কোনো ডুপ্লিকেট মান রাখা সম্ভব নয়। Set তৈরি let set = new Set([1, 2, 3, 3, 4]);
মান যোগ করা set.add(5);
মান মুছে ফেলা set.delete(2);
Variable Declaration
Variable Type Description Example
var জাভাস্ক্রিপ্টের প্রাচীনতম ভেরিয়েবল ডিক্লারেশন কীওয়ার্ড ছিলো এটি। ES6 (ECMAScript 2015) এর আগে var ব্যবহারের প্রচলন ছিলো। বর্তমানে তেমন ব্যবহৃত হয় না। এর বৈশিষ্ট্য হলোঃ
  • Function Scope: এটি শুধুমাত্র সেই ফাংশনের মধ্যে সীমাবদ্ধ যেখানে এটি ডিক্লার করা হয়েছে।
  • Re-declaration: একই ব্লকের মধ্যে var ভেরিয়েবল আবার ডিক্লার করা যায়।
    		• Var দিয়ে ভেরিয়েবল ডিক্লারেশন
    var name = "Tanvir"; console.log(name);
    আউটপুট: Tanvir
    Let let হলো ES6 থেকে প্রবর্তিত একটি ভেরিয়েবল ডিক্লারেশন কীওয়ার্ড। এটি var এর তুলনায় আরও নিরাপদ এবং উন্নত। এর বৈশিষ্ট্য হলোঃ
  • Block Scope: এটি কেবলমাত্র সেই ব্লকের মধ্যে সীমাবদ্ধ যেখানে এটি ডিক্লার করা হয়েছে।
  • Re-declaration: একই স্কোপে পুনরায় ডিক্লার করা যায় না।
    		• let দিয়ে ভেরিয়েবল ডিক্লারেশন let age = 25;
    console.log(age);
    আউটপুট: 25
    const const হলো এমন ভেরিয়েবল ডিক্লারেশন, যা একবার ডিক্লার করার পর পুনরায় মান পরিবর্তন করা যায় না। এটি সাধারণত কনস্ট্যান্ট ভ্যালু সংরক্ষণের জন্য ব্যবহৃত হয়।এর বৈশিষ্ট্য হলোঃ
  • Block Scope: এটি কেবলমাত্র সেই ব্লকের মধ্যে সীমাবদ্ধ।
  • Immutable: মান পুনরায় পরিবর্তন করা যায় না।
    		• const দিয়ে ভেরিয়েবল ডিক্লারেশন
    const pi = 3.1416; console.log(pi);
    আউটপুট: 3.1416
